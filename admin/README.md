# Ever After — Admin Panel

Foundation for the client admin dashboard.

## Features

- JWT authentication
- Profile editor
- Section visibility toggles
- Memories & gallery list (CRUD ready)
- Theme and music selection
- Live preview link

## Running

Open `admin/index.html` in a browser. The admin expects the backend running at `http://localhost:4000`.
